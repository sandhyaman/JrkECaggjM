# 旅游景点推荐系统技术文档 python200

## 项目概述

旅游景点推荐系统是基于Python+Django+Vue+MySQL技术栈开发的一套推荐系统。系统主要涉及管理员、景区管理员和用户三种角色，实现了前后端分离，为用户提供旅游景点推荐及相关管理功能。

## 技术栈

- 后端：Python + Django
- 前端：Vue
- 数据库：MySQL

## 功能模块

系统主要包括以下功能模块：

1. 用户模块
 - 注册登录
 - 景点推荐
 - 景点评论

2. 景区管理员模块
 - 景点管理
 - 评论管理

3. 管理员模块
 - 用户管理
 - 景区管理
 - 系统设置

## 系统角色

- 管理员：负责整个系统的管理和维护
- 景区管理员：负责所管理景区的信息更新和评论管理
- 用户：使用系统进行景点浏览和评论

## 运行环境

- 后端：Python 3.8+，Django 3.x，MySQL 5.7+
- 前端：Node.js 14+，Vue 2.x
